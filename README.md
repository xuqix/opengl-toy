opengl-toy: just do something with opengl!
========================================

**Required:** opengl 3.3+

**External module required:** 

* glfw-3.2(simple window manager) 
* glm-0.9.7.6(a math library for opengl) 
* SOIL(simple opengl image library)

Platform
--------

* OSX

Compile 
-------

    mkdir build
    cmake ..
    make
